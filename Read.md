# Credit Card Fraud Detection using Machine Learning

## Project Overview
This project predicts whether a credit card transaction is Genuine or Fraud using Machine Learning.

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Flask
- HTML
- CSS
- Joblib

## Machine Learning Algorithm
- Logistic Regression

## Model Performance
- Accuracy: 99.91%
- AUC Score: 0.9758

## Features
- Data Preprocessing
- Feature Scaling
- Model Training
- Fraud Prediction
- Flask Web Application

## Dataset
Credit Card Fraud Detection Dataset (Kaggle)
